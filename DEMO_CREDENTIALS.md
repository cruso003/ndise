## Permission-Based Access

The system uses **granular permissions** instead of simple roles. Each user has specific permissions in the format `resource:action`.

### Permission Examples

- `nsa:read` - Access NSA Operations Center (intelligence)
- `dashboard:read` - View executive dashboard
- `enrollment:read` - View enrollment data
- `enrollment:write` - Create/edit enrollments
- `border:read` - View border control data
- `border:write` - Process travelers, manage watchlist
- `police:read` - View police portal
- `police:write` - Manage cases, investigations
- `agency:read` - View agency services
- `reports:export` - Export reports
- `analytics:read` - Access analytics
- `audit:read` - View audit logs
- `users:manage` - Manage user accounts
- `system:manage` - System configuration

## Architecture Notes

### Platform Separation

**`ndise-web` (Current Application)** = Government Officers & Partners ONLY
- ‚úÖ NSA Operations Center
- ‚úÖ Executive Dashboard
- ‚úÖ Border Control
- ‚úÖ Enrollment
- ‚úÖ Police Portal
- ‚úÖ Agency/Partner Services

**Separate Applications (Not in this demo):**
- `ndise-citizen-mobile` - React Native app for 4.5M citizens
- `ndise-citizen-portal` - Optional web portal for citizens
- Citizens do NOT use `ndise-web` - they get separate consumer apps

### AI Integration

All dashboards are powered by AI intelligence layer:
- **8 AI Capabilities:** Duplicate detection, pattern recognition, relationship mapping, predictive analytics, anomaly detection, smart consolidation, natural language queries, risk scoring
- **Real-time Intelligence:** Continuous monitoring and alerts
- **Human-in-the-Loop:** AI recommends, humans decide
- **Explainable AI:** Confidence scores and reasoning shown

### Agency Integration

15+ agencies monitored in real-time:
- Civil Registry (NIR), Immigration, Police (LIPS), Tax Authority, NEC, LTA, DVA, Health Ministry, and more
- Real-time status: Online/Degraded/Offline
- Performance metrics: Response time, success rate, uptime
- Data sync tracking: Today/Week/Month volumes
